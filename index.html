⚙ Konfigurasi Portal
Tempel URL Google Apps Script Web App di bawah.
URL berbentuk: https://script.google.com/macros/s/.../exec
Langkah deploy Apps Script:
1. Buka Spreadsheet → Extensions → Apps Script
2. Paste isi Code.gs yang sudah diunduh
3. Jalankan fungsi initDefaultData (1x saja)
4. Klik Deploy → New Deployment → Web App
   · Execute as: Me
   · Who can access: Anyone
5. Salin URL → tempel di tab "API Backend"

Update setelah ubah Code.gs:
Deploy → Manage Deployments → Edit → New version → Update

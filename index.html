⚙ Konfigurasi Database Global
Google Spreadsheet sebagai backend via Apps Script Web App. Setiap kartu juga bisa punya link Spreadsheet sendiri.
  1. Buat Spreadsheet baru, beri nama DB Portal TPI
  2. Buat sheet apps dengan kolom: id | name | url | dbUrl | dbLabel | desc | icon | color | active
  3. Buka Extensions → Apps Script, paste script (tombol bawah)
  4. Deploy → New Deployment → Web App, akses: Anyone
  5. Salin URL Deployment → tab "Hubungkan"
